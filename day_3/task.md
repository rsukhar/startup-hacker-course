# День 3. Знакомство с Vue3 (события, связывания)

Темы:

- обработка событий
- привязка классов и стилей
- работа с данными формы
- v-model

## Практическое задание

В этом уроке мы сделаем рейтинг книг интерактивным.

Цель задания:

- научиться выводить списки в шаблоне vue
- освоить работу с реактивными переменными
- научиться привязывать классы и стили
- освоить работу с формами

## Порядок выполнения

1. Сделай звездочки интерактивными: звезда, на которую кликнул пользователь и все звезды до нее, загораются желтым цветом. Остальные — становятся серыми.
2. Пользователь может задавать рейтинг книги только один раз. После выбора рейтинга кнопки блокируются.
3. Добавь в верхнем углу обложки большую звезду со значением рейтинга. Если рейтинга нет — серую звезду с прочерком
4. Добавьте кнопку «Добавить», по нажатию на которую открывается форма создания новой книги
5. Форма должна содержать следующие поля:

- название (text)
- описание (textarea)
- обложка (text)
- 18+ (чек-бокс)
- жанр (выпадающий список)

6. После добавления книги, форма должна быть очищена
7. Добавь кнопку «Отменить», которая очищает и закрывает форму.
