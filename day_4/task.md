# День 4. Компоненты Vue3

Темы:

- компоненты vue.js
- хуки жизненного цикла
- слоты
- асинхронные компоненты

## Практическое задание

В этом уроке мы выделим части кода в отдельные компоненты, чтобы их можно было переиспользовать в других частях приложения.

Цель задания:

- научиться создавать и переиспользовать компоненты
- научиться работать с modelValue
- научиться генерировать события из дочерних компонентов в родительские
- освоить работу со слотами

## Порядок выполнения

1. Перенеси функциональную и визуальную часть карточки с книгой в отдельный компонент BookCard.vue
2. Замени соответствующий код приложения на `<BookCard />`
3. Создай computed-свойство **_rated_**, которое равно true у книг с одной и более звезд
4. Добавь атрибут book, который принимает объект с книгой
5. компонент должен «испускать» два события:

- edit — редактирование карточки. По нему в родителе нужно открывать форму редактирования книги.
- delete — удаление карточки. При этом, в родителе нужно реализовать удаление книги.

6. вынеси форму создания книги в отдельный компонент BookForm.vue
7. он должен получать данные в modelValue
8. при каждом изменении данных форма должна эммитить событие **_update:modelValue_**
9. при отмене ввода генерируется событие cancel

10. создай компонент модального окна Dialog.vue
11. содержимое окна должно передаётся в слот
12. добавь атрибут title для текста заголовка и именованный слот title для более сложных форма контента в заголовке
